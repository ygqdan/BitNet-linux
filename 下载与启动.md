# bitnet-linux 下载与启动（x86 + hf-mirror）

## 1) 下载并转换 Instruct 模型（方案 B）

```bash
cd "/home/srbank/code/BitNet－linux"
source venv-bitnet/bin/activate
export HF_ENDPOINT=https://hf-mirror.com
python setup_env.py -hr tiiuae/Falcon3-3B-Instruct-1.58bit -q i2_s
```

## 2) 启动推理（更稳中文参数）

```bash
cd "/home/srbank/code/BitNet－linux"
# 解决 llama-cli 启动时报 libllama.so / libggml.so not found
export LD_LIBRARY_PATH="$PWD/build/src:$PWD/build/ggml/src:$LD_LIBRARY_PATH"
./build/bin/llama-cli \
  -m models/Falcon3-3B-Instruct-1.58bit/ggml-model-i2_s.gguf \
  -p "你是一个中文技术助手。请用简洁中文回答：Linux 的三个核心特点是什么？" \
  -n 180 \
  --temp 0.2 \
  --top-k 20 \
  --top-p 0.8 \
  --repeat-penalty 1.15 \
  --repeat-last-n 128 \
  --min-p 0.1 \
  --no-warmup
```

## 3) 参数预设（按需要切换）

- 最稳：`--temp 0.1 --top-k 15 --top-p 0.75`
- 均衡：`--temp 0.2 --top-k 20 --top-p 0.8`
- 更自然：`--temp 0.3 --top-k 30 --top-p 0.85`

## 4) 可选清理（释放空间）

```bash
rm -f "/home/srbank/code/BitNet－linux/models/Falcon3-3B-Instruct-1.58bit/ggml-model-f32.gguf"
```

## 6) Qwen2.5-7B 中文增强版（Q4_K_M，推荐先测试）

### 6.1 下载并合并 GGUF（可能较慢）

```bash
cd "/home/srbank/code/BitNet－linux"
export HF_ENDPOINT=https://hf-mirror.com

mkdir -p "models/Qwen2.5-7B-Instruct"

# 先下载两个分片（split）
huggingface-cli download Qwen/Qwen2.5-7B-Instruct-GGUF \
  --include "qwen2.5-7b-instruct-q4_k_m-00001-of-00002.gguf" \
  --local-dir "models/Qwen2.5-7B-Instruct"

huggingface-cli download Qwen/Qwen2.5-7B-Instruct-GGUF \
  --include "qwen2.5-7b-instruct-q4_k_m-00002-of-00002.gguf" \
  --local-dir "models/Qwen2.5-7B-Instruct"

# 合并分片
./build/bin/llama-gguf-split --merge \
  "models/Qwen2.5-7B-Instruct/qwen2.5-7b-instruct-q4_k_m-00001-of-00002.gguf" \
  "models/Qwen2.5-7B-Instruct/qwen2.5-7b-instruct-q4_k_m.gguf"
```

### 6.2 启动推理（中文测试）

```bash
cd "/home/srbank/code/BitNet－linux"

# 解决 llama-cli 启动时报 libllama.so / libggml.so not found（本次会话生效）
export LD_LIBRARY_PATH="$PWD/build/3rdparty/llama.cpp/src:$PWD/build/3rdparty/llama.cpp/ggml/src:$LD_LIBRARY_PATH"

./build/bin/llama-cli \
  -m "models/Qwen2.5-7B-Instruct/qwen2.5-7b-instruct-q4_k_m.gguf" \
  -p "你是一个中文技术助手。请用简洁中文回答：中国有几个省？" \
  -n 220 \
  --temp 0.2 \
  --top-k 30 \
  --top-p 0.85 \
  --repeat-penalty 1.1 \
  --repeat-last-n 128 \
  --min-p 0.05 \
  --no-warmup
```

### 6.3 如果输出仍出现“乱码/��”：优先检查终端编码

你可以先在运行前设置 UTF-8（通常 Cursor 终端/系统设置本身也是 UTF-8，但可手动兜底）：

```bash
export LANG=zh_CN.UTF-8
export LC_ALL=zh_CN.UTF-8
```
